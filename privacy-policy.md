# Genai - Privacy Policy

Last updated: April 7, 2026

## 1. Information Collection
Genai does not collect personal information from viewers. The application
operates autonomously to create and publish news content.

## 2. Third-Party Services
Genai uses the following third-party services:
- YouTube Data API (Google Privacy Policy applies)
- Instagram Graph API (Meta Privacy Policy applies)
- TikTok Content Posting API (TikTok Privacy Policy applies)

## 3. Data Storage
All application data is stored locally on the operator's machine. No user
data is transmitted to external servers beyond the publishing platforms.

## 4. Cookies
Genai does not use cookies or tracking technologies.
